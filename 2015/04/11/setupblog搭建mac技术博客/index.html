setUpBlog搭建Mac技术博客

http://hexo.io

安装brew

执行命令

- $ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

安装wget

执行命令

- $ brew install wget

安装nvm(node.js的版本管理工具)

- $ wget -qO- https://raw.github.com/creationix/nvm/master/install.sh | sh

添加如下内容到~/.zshrc 配置文件

- [ -s "/Users/`users`/.nvm/nvm.sh" ] && . "/Users/`users`/.nvm/nvm.sh" # This loads nvm

安装node.js(通过nvm安装)

- $ nvm install 0.10

安装hexo

- $ npm install -g hexo

创建hexo文件夹，并安装hexo组件

- $ hexo init hexo
  $ cd hexo
  $ npm install

查看本地

输⼊入如下命令后,打开浏览器,并输⼊入localhost:4000来查看

- $ hexo g
  $ hexo s

登录github.com注册github账号

创建与github同名的repository

eg : github账号名位calmseapeng,则创建calmseapeng.github.io

部署到github上

如下所示编辑_config.yml

deploy: 
        type: github
        repository: git@github.com:calmseapeng/calmseapeng.github.io
        branch: master

输入如下命令，完成到github的部署，之后打开浏览器并输入calmseapeng.github.io来查看

$ hexo g
$ hexo d

上传文件到blog

在hexo目录下，创建新文件

eg : 创建新文件test

$ hexo new test

默认创建的新文件后缀问 .md文件，存在hexo/source/_posts/目录下

使用MOU打开新建的文件并完成编辑

$ mou source/_posts/test.md

将新建的文件声称网页

$ hexo g

上传到blog

$ hexo d

登录blog查看是否上传成功

http://calmseapeng.github.io

总结

搭建过程中所遇到的问题

根据官网所述步骤应该改改
安装 Node.js
cURL: $ curl https://raw.github.com/creationix/nvm/master/install.sh | sh
Wget: $ wget -qO- https://raw.github.com/creationix/nvm/master/install.sh | sh
安装完成后，重启终端并执行下列命令即可安装 Node.js
$ nvm install 0.10
安装 Hexo
所有必备的应用程序安装完成后，即可使用 npm 安装 Hexo
$ npm install -g hexo-cli
创建hexo文件夹，并安装hexo组件
$ hexo init hexo
$ cd hexo
$ npm install
添加如下内容到~/.zshrc 配置文件
vi ~/.zshrc
在.zshrc文件中添加下列内容
export PATH=$PATH:~/.nvm/v0.10.28/bin
[ -s "/Users/`users`/.nvm/nvm.sh" ] && . "/Users/`users`/.nvm/nvm.sh" # This loads nvm

部署到github上

如下所示编辑_config.yml
vi _config.yml
deploy: 
        type: git
        repository: https://github.com/CalmSeaPeng/CalmSeaPeng.github.io.git
        branch: master
输入如下命令，完成到github的部署
$ hexo g
$ hexo d
但是$ hexo d
    ERROR Deployer not found: github

怎么解决这样的问题呢？

首先输入命令：npm install hexo-deployer-git --save
            hexo clean

但是又出现错误

FATAL Something's wrong. Maybe you can find the solution here: http://hexo.io/docs/troubleshooting.html
Error: error: RPC failed; result=55, HTTP code = 200
HTTP code = 200表示已经是请求成功了，不用担心，这只是当前网速不给力造成的，等待一会重新输入
$ hexo d 就OK了！
